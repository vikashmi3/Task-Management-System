# Task Management System

I am creating full stack demo app using Backend API (Node.js + TypeScript) Which is mandatory and Frontend (Next.js + TypeScript).

Designed and developed a production-ready full-stack application with a strong focus on backend API architecture, scalability, and type safety using TypeScript across the entire stack.

## Features

- User authentication (register/login) with JWT tokens
- CRUD operations for tasks
- Task filtering and search
- Responsive design
- Token refresh mechanism

## Tech Stack

**Backend:**
- Node.js + TypeScript
- Express.js
- Prisma ORM
- SQLite database
- JWT authentication
- bcrypt for password hashing

**Frontend:**
- Next.js 14 (App Router)
- TypeScript
- React Context for state management
- CSS for styling
- React Hot Toast for notifications
